# GSLIMP.AL - Higienização e Lavagem a Seco

Este projeto é um site para a GSLIMP.AL, uma empresa especializada em higienização e lavagem a seco. O site apresenta os serviços, informações de contato e uma galeria de resultados.

## Funcionalidades

*   **Menu de Navegação Responsivo:** Adapta-se a dispositivos móveis.
*   **Galeria de Imagens (Carrossel):** Exibição de fotos de trabalhos realizados utilizando Bootstrap.
*   **Seções Informativas:** Áreas dedicadas a "Serviços" e "Sobre".
*   **Integração com YouTube:** Vídeo demonstrativo incorporado.
*   **Contato Direto:** Botão flutuante para contato via WhatsApp e link para o Instagram.

## Tecnologias Utilizadas

*   HTML5
*   CSS3
*   JavaScript
*   [Bootstrap 4](https://getbootstrap.com/)
*   [W3.CSS](https://www.w3schools.com/w3css/)
*   [Font Awesome](https://fontawesome.com/)
*   Google Fonts (Poppins)

## Estrutura do Projeto

A estrutura de arquivos do projeto é a seguinte:

```
.
├── Gslimp/
│   ├── img/            # Imagens do projeto
│   ├── index.html      # Página principal
│   ├── script.js       # Script para o menu mobile
│   └── style.css       # Estilos personalizados
└── README.md           # Documentação do projeto
```

## Como Executar

Este é um site estático. Para visualizá-lo, basta abrir o arquivo `Gslimp/index.html` em qualquer navegador web moderno.

Não é necessário instalação de dependências ou servidor local.
